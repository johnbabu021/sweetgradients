# SWEETGRADIENTS


###   Features
* almost 300 gradients
* framework ReactJS
* dark and light modes


### Installation
1.Clone the repository

```bash
https://github.com/johnbabu021/sweetgradients.git
```
2.Change   your directory
```bash
cd sweetgradients
```
3.Install all packages
```bash
npm install
```
4.run the project

```bash     
npm start
```


All set



###   Contribution

*   switch  to gradients.json
```bash
cd src/constants
```

*   create a new object with name and color

```bash
{
    "name":"your_gradient_name",
    "colors":["#YOUR_gradient","YOUR_gradient","YOUR_gradient",....]
}
```


* send a pull Request
